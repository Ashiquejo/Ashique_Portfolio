## ⚙️ Tech Stack

* ⚛️ React + Vite
* 🎨 Tailwind CSS
* 🎞️ Framer Motion
* 🌐 React Router
* 📩 EmailJS (contact integration)
* 🧠 Gemini API (AI assistant – in progress)
* ⚡ GSAP (advanced animations)

